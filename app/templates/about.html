About Pleyt

Pleyt, pronounced like Plate, is an app that compares yelp reviews to sentiment data from twitter. It uses business listing APIs from Yelp and Twitter, and uses the AlchemyAPI for sentiment analysis. It is written in Python and uses Flask for the backend and Bootstrap for the frontend.

How to use Pleyt

Regions can be queried just like one would perform a yelp search. Businesses can then be selected, and sentiment data is gathered and presented.
